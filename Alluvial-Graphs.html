The ggalluvial package is a ggplot2 extension for producing alluvial plots. In case you need to remember what an Alluvial plot is: Alluvial plots use variable-width ribbons and stacked bar plots to represent multi-dimensional or repeated measures data with categorical or ordinal variables Two types - Alluvial (Wide) Format & Lodes (Long) Format.

There are five essential components to this package: 1.AXIS: A dimension (variable) align which the data are vertically grouped at a fixed horizontal position 2.ALLUVIUM: Horizontal (x-) lines called alluvia span the width of the plot 3.STRATUM: The groups at each axis are depicted as opaque blocks called strata (columns) 4.LODE: The alluvia insersect the strata at lodes 5.FLOW: The segments of the alluvia between pairs of adjacent axes are flows

grouped_data <- Alluvial.Plot.Test.1 %>% 
  group_by(PbEl04Residue, C2Residue) %>%
  summarise(Freq = n())
## `summarise()` has grouped output by 'PbEl04Residue'. You can override using the
## `.groups` argument.
#ggplot is the main function that renders the plot
ggplot(as.data.frame(grouped_data), 
       aes(y = Freq, 
         axis1 = PbEl04Residue, 
         axis2 = C2Residue,)) +
geom_alluvium(aes(fill = PbEl04Residue)) +
  #geom_stratum receives a dataset of the horizontal 
  #(x) and vertical (y) positions of the strata of an alluvial plot.
  #It plots rectangles for these strata (columns) of a provided width
  geom_stratum(aes(fill = PbEl04Residue)) +
  geom_text(stat = "stratum", aes(label = after_stat(stratum))) +
#The "strat = stratum" argument here references
#that the text present here is to be based on 
#stratum (columns) regarding the grouped data data frame
  ggtitle("Interactions between PbEL04 Residues and C2 Residues") +
  theme_void()
